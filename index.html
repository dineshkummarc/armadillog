Welcome!



Armadillog is a tool which makes it easier and faster to go through a large amount of text, e. g. application logs.

Input a piece of text or a file, apply a set of filters and check out the result.

You can narrow the output down to the point where you see only what is relevant to your current need.



It's still a work in progress, so please feel free to send me any bug reports or feature requests.
